# 거북이 게임

### 게임 설명
<span style="color:red">**빨간**</span> 거북이를 조종하여 <span style="color:blue">**파란**</span> 거북이를 잡는 게임입니다!

- **제한 시간:** 60초
- **목표:** <span style="color:blue">**파란**</span> 거북이를 최대한 많이 잡기

### 점수에 따른 실력 구분
- **상 (8점 이상)**
- **중 (5점 이상)**
- **하 (그 이하)**

### 주의 사항
1. **<span style="color:blue">파란</span> 거북이를 잡았을 시**, 두 마리 거북이의 위치가 모두 **랜덤**으로 배정됩니다.
2. **<span style="color:blue">파란</span> 거북이가 게임 화면 상의 경계에 도달할 시**, <span style="color:blue">파란</span> 거북이의 위치가 **랜덤**으로 배정됩니다.
